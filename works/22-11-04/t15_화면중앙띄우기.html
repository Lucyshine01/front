화면중앙에 새창띄우기(부모창)

텍스트박스에 윈도우 가로크기와 세로크기를 입력받아서 버튼을 클릭하면 화면 중앙에 사용자가 지정한 크기로 새창(t15_child.html)을 띄우시오.

윈도우 가로 크기 :

윈도우 세로 크기 : 

  : 새창에 사용되는 이름(name)?
      _blank : 새창(기본값), _parent : 부모프레임에 열린다. _top : 로드된 프렝미셋을 다시대체. _self:자신창을 띄운다.
  : 새창에 사용되는 옵션(option)?
      width : 창의 폭
      height : 창의 높이
      left : x축 위치
      top : y축 위치

      --IE에서만 작동, 모던 웹브라우저(크롬,파이어폭스,오페라 등등 안됌)
      fullscreen=(yes/no/auto) : 전체화면모드(지원x)
      titlebar=(yes/no/auto) : 제목표시줄의 표시여부 결정(지원x)
      location=(yes/no/auto) : 주소표시줄의 출력여부를 결정(지원x)
      menubar=(yes/no/auto) : 메뉴표시줄의 표시여부 결정(북마크바)(지원x)
      resizable=(yes/no/auto) : 창의 크기를 제어 결정(지원x)
      scrollbars=(yes/no/auto) : 스크롤바의 표시여부 결정(지원x)
      toolbar(yes/no/auto) : 툴바(도구상자)의 표시여부 결정(지원x)
